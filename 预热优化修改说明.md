1. `model/MultiCameraInterface.py`
2. `record-eggs-second/model/MultiCameraInterface.py`

### 4. 预热方法替换为快速验证

**修改前：**
```python
def _warmup_camera(self, camera_idx):
    """预热摄像头（改进版）"""
    # 清空缓冲区5次
    for _ in range(5):
        cap.grab()
    time.sleep(0.2)
    
    # 尝试读取15帧来预热
    for i in range(15):
        ret = cap.grab()
        if ret:
            ret, frame = cap.retrieve()
            # ... 复杂的预热逻辑
        time.sleep(0.1)
```

**修改后：**
```python
def _quick_camera_test(self, camera_idx):
    """快速验证摄像头可用性（只读取一帧）"""
    # 清空缓冲区2次
    for _ in range(2):
        cap.grab()
    
    # 只读取一帧验证可用性
    ret, frame = cap.read()
    if ret and frame is not None:
        print(f"摄像头 {camera_idx} 验证成功，成功读取 1 帧")
        return True
    else:
        print(f"摄像头 {camera_idx} 验证失败，无法读取帧")
        return False
```

- **线程启动等待时间**：从 2秒 减少到 0.4秒

### 总启动时间优化
- **修改前总等待时间**：约 8.5秒

- **优化效果**：启动速度提升约 **85%**
